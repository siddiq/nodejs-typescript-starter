# Node.js + TypeScript + Jest Starter 🚀

A simple boilerplate for a Node.js project with TypeScript and Jest.

## ⚡ Setup

```sh
npm install
```

## ▶️ Run

```
npm start
```

## 🧪 Test

```
npm test
```

## 🔧 Build

```
npm run build
```

## ▶️▶️ Run in Production

```
npm run start:prod
```

Happy coding! 🚀
